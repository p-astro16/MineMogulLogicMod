Features

Everything MML adds to your MineMogul factory session.

📊

Factory HUD

A full dark-themed canvas overlay with tabs for Machines, Belts, Sales and Settings. Draggable, closable, and live-updating while you play.

F54 Tabs
⚡

Throughput Tracker

Tracks items-per-minute on every conveyor belt in your factory. See what each belt carries and at what rate — no guesswork needed.

Belts Tab
⚠️

Bottleneck Detector

Automatically flags the slowest belts relative to your factory's average. Highlighted in red, sorted low→high — jump straight to the problem.

Belts Tab
🔧

Splitter Wrench

A buyable tool from the in-game shop. Left-click any RollerSplitter to set routing ratios per resource type. Right-click opens a global view of every splitter on the map.

New · v1.0 Shop Item
🏷️

Belt Scanner

A purchasable tool from the shop (Tools category). Uses the ResourceScanner model. When equipped, floating items/min labels appear above every nearby belt — color-coded by throughput.

Shop ItemNew · v1.0
🔬

Ore Analyser Upgrade

Extends the built-in scanner with sell value per unit, processing history, and sorter stats for nearby ore nodes.

HUD Tab
💰

Sales Tracker

Tracks session earnings, top-selling resources, and calculates your current €/min revenue rate with a live summary in the HUD.

Sales Tab
🖥️

Terminal Integration

Open the Factory HUD from any in-game Computer Terminal via the Factory Overview interaction — no need to remember hotkeys.

Getting started

Installation

The installer handles BepInEx and the mod automatically.

1

Download the latest release

Grab MML_v1.0.0.zip from the GitHub Releases page and extract it anywhere.

2

Run MML_Installer.exe

Launch the installer. Your game path is auto-detected via Steam — or click Browse to set it manually.

3

Click Install

Copies BepInEx and MML.dll into the correct folders. Launch the game — the mod activates immediately.

4

Press F5 in-game

The Factory HUD opens. Visit the in-game shop terminal to buy the Splitter Wrench from the Tools category.

Manual install

Download BepInEx 5.4.23 x64 and extract it into your MineMogul game folder.
Copy MML.dll into <MineMogul>/BepInEx/plugins/
Launch the game. Config is created at BepInEx/config/com.minemogul.mml.cfg
Press F5 to open the HUD, F6 for belt counters.

Uninstall: run the installer and click Uninstall, or delete BepInEx/plugins/MML.dll.
