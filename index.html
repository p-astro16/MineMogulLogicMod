What's inside

Features

Everything MML adds to your MineMogul factory session.

📊

Factory HUD

Press F5 to open a live dark-themed overlay with five tabs: Machines, Belts, Sales, Ore Analyser, and Settings. All data updates in real-time while you play.

HUD · F5
⚙️

Throughput Tracker

Tracks items-per-minute on every conveyor belt in your factory. See what each belt is carrying and at what rate — no guesswork needed.

HUD Tab
⚠️

Bottleneck Detector

Automatically flags the slowest belts relative to your factory's average. Jump straight to the problem without walking every belt manually.

HUD Tab
🔧

Splitter Wrench

A buyable tool item. Left-click any RollerSplitter or Smart Splitter to configure its routing ratios per resource type. Right-click opens a global overview of every splitter on the map.

New · v1.0.0 Tool Item
🏷️

Belt Item Counter

Press F6 to toggle floating IPM labels on belts near you. Numbers update live and color-code by saturation.

Toggle · F6
🔬

Ore Analyser Upgrade

Extends the built-in scanner readout with sell value per unit, processing history, and a quick-compare chart for nearby ore nodes.

HUD Tab
💰

Sales Tracker

Tracks your session earnings, shows top-selling resources, and calculates your current revenue rate per minute.

HUD Tab
🖥️

Terminal Integration

Open the Factory HUD from any in-game Computer Terminal via the Factory Overview interaction — no need to remember the hotkey.

New · v1.0.0

Getting started

Installation

Use the included installer — it handles BepInEx and the mod automatically.

1

Download the installer

Download MML_v1.0.0.zip from the releases page and extract it anywhere.

2

Run MineMogulModInstaller.exe

Launch the installer. On the landing page you'll see feature highlights and a version badge.

3

Select your MineMogul folder

Click Browse and navigate to the folder where MineMogul.exe lives. The installer validates the path automatically.

4

Click Install

The installer copies BepInEx and MML.dll into the correct folders. Launch the game — the mod is active immediately.

5

In-game: press F5

The Factory HUD opens. Visit the shop terminal in-game to buy the Splitter Wrench from the Tools category.
